# Code of Conduct

nSealr contributors are expected to keep collaboration professional,
technical, and respectful.

## Expected Behavior

- Focus on facts, evidence, and reproducible results.
- Challenge weak technical claims directly but respectfully.
- Document uncertainty instead of overstating confidence.
- Keep reviews actionable.
- Respect privacy and responsible disclosure for security issues.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Public disclosure of security issues before maintainers have had a reasonable
  chance to respond.
- Knowingly publishing false security or maturity claims.
- Introducing license-incompatible code without disclosure.

Maintainers may remove comments, issues, pull requests, or access when behavior
creates risk for contributors or users.
