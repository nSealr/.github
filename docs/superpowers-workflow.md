# Superpowers Workflow

Superpowers is a development workflow for agents. It is not a runtime dependency
of nSealr.

## Required Use

- Use planning skills before large implementation work.
- Use an isolated branch or worktree for substantial changes.
- Use test-driven development where behavior changes.
- Use verification-before-completion before any success claim.
- Use branch finishing workflow before merge, PR, or cleanup.

If Superpowers skills are not loaded in the current Codex session, follow the
same process manually and restart the session before long-running work.
