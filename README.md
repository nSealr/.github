# nSealr GitHub Profile

This repository contains organization-level GitHub metadata for nSealr.

nSealr is a non-profit open-source hardware and software project for Nostr
signing devices. The project publishes reproducible reference designs, firmware,
companion software, protocols, test vectors, and research notes.

## Quality Baseline

Run the organization metadata verification loop with:

```sh
make ci
```
